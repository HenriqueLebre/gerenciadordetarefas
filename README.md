# Gerenciador de Tarefas!

Aplicativo de gerenciamento de tarefas com um temporizador! 
Realizado com Vue3.

Aplicativo do curso de Vue3 ensinado pela excelente professor Vinicius, projeto com algumas adaptações.

npm install
